# 🌯 Shawarma Bot — Инструкция по запуску

## Файлы в папке:
- `bot.py` — основной код бота
- `credentials.json` — ключ Google (уже внутри)
- `requirements.txt` — библиотеки
- `Procfile` — для Railway

---

## Шаг 1 — Загрузи на GitHub

1. Зайди на **github.com** → зарегистрируйся (если нет аккаунта)
2. Нажми **"New repository"**
3. Назови `shawarma-bot` → **Create repository**
4. Загрузи все 4 файла (bot.py, credentials.json, requirements.txt, Procfile)

---

## Шаг 2 — Запусти на Railway

1. Зайди на **railway.app**
2. Войди через GitHub аккаунт
3. Нажми **"New Project"** → **"Deploy from GitHub repo"**
4. Выбери `shawarma-bot`
5. Railway сам установит всё

---

## Шаг 3 — Добавь сотрудников в бота

Открой `bot.py`, найди раздел CASHIERS и COURIERS:

```python
CASHIERS = {
    OWNER_ID: "Мурад (тест)",
    # 111111111: "Ильхам",   ← раскомментируй и вставь реальный ID
    # 222222222: "Сальма",
    # 333333333: "Милена",
}
```

Чтобы узнать Telegram ID сотрудника:
- Попроси его написать боту @userinfobot команду /start
- Он получит свой ID

---

## Команды бота:

- `/start` — главное меню
- `/cancel` — отменить текущее действие

## Что бот умеет:

✅ Отчёт кассира — все поступления по источникам  
✅ Данные по курьерам (часы + суммы)  
✅ Расходы (карта + наличные)  
✅ Аванс — запрос + одобрение одной кнопкой  
✅ Сводка за день (для владельца)  
✅ Всё автоматически в Google Sheets (лист БОТ_Касса)  

---

## Google Sheets — что появится:

Бот создаст 2 новых листа в твоей таблице:
- **БОТ_Касса** — все ежедневные отчёты
- **БОТ_Авансы** — история авансов

Твои старые листы НЕ трогаются!
